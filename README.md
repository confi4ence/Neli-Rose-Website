# Nelirose Retirement Home Website

This project is a responsive website created for **Nelirose Retirement Home**, designed to provide information about the home, its services, and the care offered to residents.



## Project Overview

The Nelirose website was developed to create a warm, welcoming, and informative online space where families and potential residents can learn more about the retirement home. 

The focus of the design is on clarity, accessibility, and trust.



## Features

- Responsive design (mobile, tablet, and desktop friendly)
- Clean and calming user interface
- Easy-to-use navigation
- Informational sections such as:
  - Home
  - About Us
  - Services / Care Offered
  - Contact Information
- Images to create a welcoming feel



## Technologies Used

- HTML5
- CSS3
- JavaScript



## Purpose of the Project

This project was created to:

- Provide an online presence for a retirement home
- Share important information with families and visitors
- Practice front-end web development skills
- Design a user-friendly and accessible website
